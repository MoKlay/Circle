О приложении

    Данное приложение расчитанно в целях работы с данными в количестве десяти окружностей. Расчитывается среднее значение радиуса и выводятся координаты введенных окружностей, которые больше среднего радиуса.

Руководство по использованию приложения

В главном окне:
    - С помощью кнопки Добавить пользователь добавляет данные в массив объектов, который ввел  в текстовые поля;
    - С помощью кнопки Удалить пользователь удаляет окружность, выбранный по индексу;
    - При нажатии на кнопку Удалить все удаляется весь набранный пользователем/загруженный из файла список;
    - Когда пользователь нажимает кнопку Сохранить, то сохраняется данные в CSV файле в выбранном пользователем пути сохранения;
    - Когда же нажимаете кнопку Загрузить, то выбранный файл CSV загружается в массив объектов и выводится в интерфейсе;
    - При заполнении списка показывается кнопка Результата, которая же открывает окно с результатом выполнения приложения, и скрываются текстовые поля.

    Также при заполненном списке, когда пользователь нажимает кнопку удалить или удалить все, кнопка Результат скрывается и отображается текстовые поля.

В окне с результатом:
    На окне с результатом записывается в текстовое поле результаты выполнения и также можно сохранить эти результаты в отдельный текстовой файл.
